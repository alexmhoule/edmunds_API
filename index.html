My Edmunds API Assignment

This application will allow you to look up the specifications of two vehicles to compare using the Edmunds API to retrieve data.

  1. Simply select the make of vehicle from the drop down menu.
  2. Next the model drop down will appear, select the desired model.
  3. With the make and model selected, you will then select the year of the vehicle.
  4. The final step will be selecting the style of that vehicle
  5. Now view the data for both vehicles and compare
